# Personal Portfolio Website

## Project Description
A responsive personal portfolio website showcasing skills, projects, and contact information. Built using HTML5, CSS3, and JavaScript following modern web design principles.

## Features
- Responsive design (mobile, tablet, desktop)
- Semantic HTML5 structure
- CSS Grid and Flexbox layouts
- Accessible navigation
- Contact form with validation
- Hover effects and animations

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## Setup Instructions
1. Clone the repository  
2. Open `index.html` in a browser  
3. Customize content and images  

## Code Structure
- `index.html` – Main HTML file  
- `css/` – Styling and responsive layout  
- `js/` – Navigation interactivity  
- `images/` – Profile and project images  

## Technical Requirements Met
- Semantic elements (`header`, `nav`, `main`, `section`, `footer`)
- External CSS files
- Responsive layout using media queries
- Grid and Flexbox
- Accessible images and navigation

## Testing
- Tested on Chrome and Edge
- Responsive on mobile and desktop
- HTML & CSS validated
